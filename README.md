## Project Setup

### Clone the repository:  
```sh
git clone https://github.com/nhhoang14/DinhClub.git
cd DinhClub
```

### Install dependencies:
```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
